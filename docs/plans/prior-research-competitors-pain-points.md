**Key findings (2026-04-16):**

1. **Market leaders:** Slate by Technolutions ($30k+/yr), Salesforce Education Cloud (most expensive), Ellucian CRM, Element451 (AI-first), TargetX
2. **Average university uses 35 different tools; ~20% use 70+** — massive fragmentation and data silos
3. **Biggest pain points:** Data silos between systems, manual processes (could save 13hrs/week/staff with automation), staff burnout, slow communication losing leads, vendor lock-in with expensive multi-year contracts, difficult data migration (47% cite as barrier to switching)
4. **Thai context:** TCAS is centralized but frustrating/inflexible. Universities have limited control. A platform giving them better control + data + student experience fills a real gap.
5. **What makes it irresistible:** Unified system replacing multiple tools, dramatic labor savings (70% reduction possible), predictive analytics for yield, real-time pipeline visibility, easy implementation + low switching costs, cheaper than Slate/Salesforce

**Why:** Shapes what we build for the university side — focus on consolidation, automation, pipeline visibility, and TCAS integration.

**How to apply:** When designing university features, prioritize: (1) applicant pipeline dashboard with real-time status, (2) communication automation, (3) document workflow management, (4) analytics/reporting, (5) integration with TCAS rounds. Position as "replace 5-10 tools with one platform."
